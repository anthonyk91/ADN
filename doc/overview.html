Abstract Deep Network


This package includes an implementation of the ADN algorithms, and routines for running experiments based on script files.


The software is written as a template method, where the ADN library is based around a process of making an observation, performing an action, and making updates as necessary. It is written as an OO implementation, with the intention of allowing parts to be modified in a flexible manner. This can be done by subclassing, where parts can be extended or replaced to change the design.

Overview


For a description of the algorithm see (preprint link).


The ADN library (/ADN) contains definitions for fragments (atomic and composite), and a harness for managing the lifecycle of fragments. The harness has routines for making an observation of the world, passing activations through the feature network, and making adjustments after receiving feedback.


The Point folder (/Point) extends the classes defined in the ADN library, and provides a working implementation. This operates on environments (datasets) with a set of real, int or nominal values per instance, typical of UCI/ARFF formats.


(The convolutional implementation, which operates on 2D inputs such as images, is not included as it doesn't fit together after I reworked the other parts.)
